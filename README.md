A project with a custom build using React, Redux and other popular technologies.

## First start

> Need install `dependencies`

```
npm i
```

## Local DEV

> Print command

```
npm run start:dev
```
open https://localhost:3000/

---

the project was implemented with a fake database. In the script above we will start a local server.
